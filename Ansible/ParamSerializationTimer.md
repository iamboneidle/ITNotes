# Описание
Это callback_plugin, который позволяет выводить в консоль во время установки время, которое требовалось на сериализацию переменных из инвентори.

Как подключить:

1. В файл `ansible.cfg` в поле `[defaults]` добавить `callback_plugins = callback`;
2. Поместить файл с кодом плагина в корень к `ansible.cfg` в папку `callback/`.
