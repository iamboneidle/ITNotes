# 1. Виртуальные терминалы
Можно узнать имя текущего терминала, на которой выполнен вход в системы, командой `tty`. Список всех терминальных входов пользователей - `users`, `who`, `w`.
___
# 2. Что происходит при вызове команды
1. Системный вызов `fork()`
   Создается копия процесса консоли;
2. Копия процесса выполняет команду с помощью системного вызова `exec()`;
3. Во время работы копии процесса родительский процесс находится в ожидании из-за системного вызова `wait()`;
4. По окончании работы дочерний процесс выполняет вызов `exit()`, который отсылает сигнал `SIGCHLD` оригинальному процессу, сообщая, что дочерний процесс завершен.
