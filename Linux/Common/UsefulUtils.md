# 1. Навигация и работа с файлами
- **`ls`** — показывает список файлов и директорий.
- **`cd`** — переход в другую директорию.
- **`pwd`** — показывает текущий путь директории.
- **`tree`** — отображает структуру каталогов в виде дерева.
- **`stat`** — показывает подробную информацию о файле.
- **`file`** — определяет тип файла.
- **`touch`** — создаёт новый файл или обновляет timestamp.
- **`cp`** — копирует файлы или директории.
- **`mv`** — перемещает или переименовывает файлы.
- **`rm`** — удаляет файлы или директории.
___
# 2. Просмотр и обработка файлов
- **`cat`** — выводит содержимое файла.
- **`less`** — постраничный просмотр файла.
- **`head`** — показывает первые строки файла.
- **`tail`** — показывает последние строки файла.
- **`tail -f`** — следит за файлом в реальном времени (часто для логов).
- **`wc`** — считает строки, слова и байты.
- **`sort`** — сортирует строки файла.
- **`uniq`** — удаляет повторяющиеся строки.
- **`cut`** — извлекает столбцы текста.
- **`paste`** — объединяет строки файлов.
___
# 3. Поиск и фильтрация
- **`grep`** — поиск текста по шаблону (regex).
- **`egrep` / `grep -E`** — расширенный regex поиск.
- **`find`** — поиск файлов по параметрам (имя, дата, размер).
- **`locate`** — быстрый поиск файлов по базе.
- **`which`** — показывает путь к исполняемому файлу.
- **`whereis`** — ищет бинарник, исходники и man.
___
# 4. Текстовые трансформации
- **`awk`** — мощный инструмент обработки текста и таблиц.
- **`sed`** — потоковый редактор (замена текста, фильтрация).
- **`tr`** — заменяет или удаляет символы.
- **`xargs`** — передаёт вывод команды как аргументы другой.
___
# 5. Распаковка и сжатие
- **`tar`** — создание и распаковка архивов.
- **`gzip`** — сжатие файлов.
- **`gunzip`** — распаковка gzip.
- **`zip`** — создание zip-архивов.
- **`unzip`** — распаковка zip.
___
# 6. Сеть
- **`curl`** — HTTP/HTTPS запросы, API тестирование.
- **`wget`** — скачивание файлов по URL.
- **`ping`** — проверка доступности хоста.
- **`netstat`** — информация о сетевых соединениях.
- **`ss`** — современная замена netstat.
- **`dig`** — DNS-запросы.
- **`nc` (netcat)** — диагностика TCP/UDP соединений.
___
# 7. Мониторинг системы
- **`top`** — просмотр процессов и загрузки CPU.
- **`htop`** — улучшенный интерактивный top.
- **`ps`** — показывает запущенные процессы.
- **`kill`** — завершает процесс по PID.
- **`df`** — показывает использование диска.
- **`du`** — показывает размер директорий.
- **`free`** — показывает использование RAM.
- **`uptime`** — показывает аптайм и load average.
