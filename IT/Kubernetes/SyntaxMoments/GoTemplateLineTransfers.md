# 1. Базовое поведение
Всё, что находится вне `{{ ... }}`, копируется в вывод буквально, включая все пробелы и переносы строк.
Код:
```go
Hello,
World
```
Результат:
```
Hello,
World
```
___
# 2. Управление пробелами с помощью дефиса
Это основной механизм контроля пробелов в Go templates.

|   Синтаксис   | Значение                                        | Пример                 | Результат               |
|:-------------:|:----------------------------------------------- |:---------------------- |:----------------------- |
|     `{{-`     | Обрезает пробелы и переносы **перед** действием | `Hello, {{- "World"}}` | `Hello,World`           |
|     `-}}`     | Обрезает пробелы и переносы **после** действия  | `{{"Hello" -}} World`  | `HelloWorld`            |
| `{{- ... -}}` | Обрезает с обеих сторон                         | `{{- "Hello" -}}`      | `Hello` (без окружения) |
**Важно**: Пробел после дефиса обязателен. `{{- 3}}` — это обрезка пробелов и число 3, а `{{-3}}` — это действие с числом -3.
## 2.1. Практический пример с if/else
Без trim-маркеров:

Код:
```go
{{ if .Enabled }}
  Feature enabled
{{ end }}
```
Результат:
```
[пустая строка]
  Feature enabled
[пустая строка]
```
С `trim`-маркерами:
Код:
```go
{{- if .Enabled -}}
  Feature enabled
{{- end -}}
```
Результат:
```
Feature enabled
```
(компактно, без лишних строк)

# 3. Комментарии и пробелы
Комментарии в Go templates используют синтаксис `{{/* ... */}}`. По умолчанию они тоже создают пробелы.
Код:
```go
Hello {{/* это комментарий */}} World
```
Результат:
`Hello  World` (два пробела)

Решение — использовать trim-маркеры и для комментариев:
Код:
```go
Hello {{- /* это комментарий */ }} World
```
Результат:
`Hello World` (один пробел)
## 3.1. Что есть пробел?
Определение такое же, как в Go: пробел, горизонтальная табуляция, возврат каретки и новая строка.

# 4. Спец. расширения
Существуют расширенные версии шаблонизаторов, например []() [github.com/veigaribo/template](https://github.com/veigaribo/template), которые добавляют дополнительные контроллеры пробелов:

| Символ | Назначение                                                                        |
| ------ | --------------------------------------------------------------------------------- |
| `~`    | Обрезает только пробелы, **сохраняя** переносы строк                              |
| `#`    | Более сложная логика: оставляет перенос строки, если контент был на другой строке |
## 4.1. Пример
Пример с `~`:
Код:
```go
line 1
  {{~ "line 2"}}
line 3
```
Результат:
```
line 1
line 2
line 3
```
